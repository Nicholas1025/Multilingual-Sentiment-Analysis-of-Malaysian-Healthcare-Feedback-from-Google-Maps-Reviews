Research system overview

A multilingual NLP pipeline for Malaysian healthcare feedback.

Built for the TNL6323 Natural Language Processing course project, the app brings preprocessing, sentiment modelling, aspect analysis, ranking, and retrieval-based Q&A into one polished demo surface.

Pipeline

  1. Data collection. Google Maps reviews were collected for Malaysian healthcare providers, including recovery of original review text where available.
  2. Preprocessing. The multilingual cleaning pipeline uses NLTK for English, Sastrawi for Malay, jieba for Chinese, and emoji-to-sentiment token conversion.
  3. Feature representation. Reviews are split into train, validation, and test sets with TF-IDF unigram and bigram features for baseline models.
  4. Model training. Logistic Regression, SVM, and Naive Bayes baselines are included, with a fine-tuned XLM-RoBERTa multilingual transformer for stronger inference.
  5. Aspect analysis. The app identifies healthcare aspects such as doctor, waiting time, facilities, cost, treatment, and cleanliness.
  6. Deployment. FastAPI serves the backend APIs while vanilla HTML, CSS, and JavaScript power the interface.
